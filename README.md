# Number Guessing Game (Task 2)

## Description
A simple Java console game where the user has to guess the number generated by the computer.  
The computer provides hints if the user's guess is higher or lower than the generated number.

## Features
- Random number generation
- User input with hints (higher/lower)
- Optional scoring based on attempts

## How to Run
1. Open the project in IntelliJ IDEA.
2. Navigate to `NumberGuessingGame.java`.
3. Run the program.
4. Follow the console instructions to guess the number.

## Example
Enter your guess: 50
Too low!
Enter your guess: 75
Too high!
Enter your guess: 63
Correct! You guessed the number in 3 attempts.
